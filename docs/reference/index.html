scrollConfig

scrollConfig

Type: object

Properties
transitionActive (boolean?) : whether to animate effect progress.
transitionFriction (number?) : between 0 to 1, amount of friction effect in the transition. 1 being no movement and 0 as no friction. Defaults to 0.4.
velocityActive (boolean?) : whether to calculate velocity with progress.
velocityMax (number?) : max possible value for velocity. Velocity value will be normalized according to this number, so it is kept between 0 and 1. Defaults to 1.
observeSize (boolean?) : whether to observe size changes of container . Defaults to true .
observeViewport (boolean?) : whether to observe entry/exit of scenes into viewport for disabling/enabling them. Defaults to true .
viewportRootMargin (boolean?) : rootMargin option to be used for viewport observation. Defaults to '7% 7%' .
root ((Element | Window)?) : the scrollable element, defaults to window.
wrapper (Element?) : element to use as the fixed, viewport sized layer, that clips and holds the scroll content container. If not provided, no setup is done.
container ((Element | null)?) : element to use as the container for the scrolled content. If not provided assuming native scroll is desired.
scenes (Array<ScrollScene>) : list of effect scenes to perform during scroll.
snaps (Array<SnapPoint>?) : list of scroll snap points.

ScrollScene

ScrollScene

Type: Object

Properties
start (number) : scroll position in pixels where effect starts.
duration (number?) : duration of effect in pixels. Defaults to end - start.
end (number?) : scroll position in pixels where effect ends. Defaults to start + duration.
effect (function) : the effect to perform.
pauseDuringSnap (boolean?) : whether to pause the effect during snap points, effectively ignoring scroll during duration of scroll snapping.
disabled (boolean?) : whether to perform updates on the scene. Defaults to false.
viewSource (Element?) : an element to be used for observing intersection with viewport for disabling/enabling the scene, or the source of a ViewTimeline if scene start/end are provided as ranges.

SnapPoint

SnapPoint

Type: Object

Properties
start (number) : scroll position in pixels where virtual scroll starts snapping.
duration (number?) : duration in pixels for virtual scroll snapping. Defaults to end - start.
end (number?) : scroll position in pixels where virtual scroll starts snapping. Defaults to start + duration.
